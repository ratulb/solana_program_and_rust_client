# The Rust Client

The client is a Rust CLI [program](https://github.com/ratulb/solana_program_and_rust_client/blob/main/client/src/main.rs) with a `main` function that performs five tasks:

1. [Establish a connection to the cluster](./connection.md)
2. [Set up an account to store counter program state](./account-setup.md)
3. [Check if the on-chain program has been deployed](./deployment-check.md)
4. [Send a counter increment transaction](./send-transaction.md)
5. [Query the counter account](./query-counter.md)
